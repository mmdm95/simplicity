# Simplicity Framework
A simple yet nice PHP framework

## Attention

Please run following commands before use:

```
composer install
```

```
npm install
```

then run

```
php sim webpack:clean-files
```

then

```
// for development
npm run start

// for production
npm run build
```

and then run

```
php sim webpack:move-files
```
